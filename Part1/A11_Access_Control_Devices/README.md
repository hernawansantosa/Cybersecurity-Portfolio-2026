##A11 — Discover 5 Unique Access Control Devices
This activity identifies five different access control devices observed in various environments.

1. Biometric Fingerprint Scanner
 #Description
 A fingerprint scanner verifies a user’s identity using their unique fingerprint before granting access.

 #Security Concept
 - Biometric authentication (something you are)
 - Identity verification

 #Analysis
 Fingerprint scanners reduce the risk of credential sharing or theft. 
 However, biometric data must be securely stored and encrypted to prevent misuse.

2. PIN Keypad Lock
 #Description
 A keypad lock requires entry of a numeric code to unlock a door or secured area.

 #Security Concept
 - Knowledge-based authentication (something you know)

 #Analysis
 PIN systems are simple and cost-effective but can be vulnerable to shoulder surfing or code sharing if not properly managed.

3. Turnstile Access Gate
 #Description
 A turnstile allows only one person to pass after successful credential verification.

 #Security Concept
 -Physical access control
 -Anti-tailgating mechanism
 
 #Analysis
 Turnstiles prevent multiple individuals from entering on a single authorization event, improving crowd control and security.

4. License Plate Recognition (LPR) System
 #Description
 An automated parking system that scans vehicle license plates and checks them against an authorized database before granting entry.

 #Security Concept
 - Automated identity verification
 - Authorization control
   
 #Analysis
 LPR systems provide seamless vehicle-based access control. However, accuracy and database security are critical to prevent unauthorized entry.

5. Facial Recognition Access System
 #Description
 A facial recognition system grants access by analysing a person’s facial features and matching them against stored biometric data.

 #Security Concept
 - Contactless biometric authentication

 #Analysis
 Facial recognition improves convenience and reduces physical contact, but it raises privacy concerns and requires strong data protection measures.
