# Evidence for A13 — Online Security Tools

This folder contains supporting screenshots for Activity A13.

---

## mfa_authentication.png

Screenshot showing multi-factor authentication (2-Step Verification) enabled on a Google account.

This security feature requires an additional verification step beyond a password, such as a prompt or one-time code. It enhances account security by preventing unauthorized access even if the password is compromised.

No personal or sensitive information is exposed.

---

## password_manager.png

Screenshot of a password manager showing stored account credentials and a security warning for reused passwords.

The password manager helps users securely store and manage passwords, while also identifying weak or reused passwords that may increase security risks.

Sensitive information such as email addresses and account details has been obscured.


---

## firewall_settings.png

Screenshot showing the firewall enabled on a macOS system.

The firewall monitors and controls incoming network connections, preventing unauthorized applications and services from accessing the system.
