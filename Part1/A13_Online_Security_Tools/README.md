# A13 — Discover 5 Unique Online Security Tools

This activity identifies five online security tools used to protect systems, data, and user accounts in digital environments.

---

## 1. Firewall

### Description  
A firewall monitors and controls incoming and outgoing network traffic based on predefined security rules.

### Security Concept  
- Network security  
- Traffic filtering  

### Analysis  
Firewalls act as a barrier between trusted and untrusted networks, preventing unauthorized access. They are essential for protecting systems from external attacks such as intrusion attempts and malware.

---

## 2. Antivirus Software

### Description  
Antivirus software detects, prevents, and removes malicious software such as viruses, worms, and trojans.

### Security Concept  
- Threat detection  
- Malware protection  

### Analysis  
Antivirus tools use signature-based and behavior-based detection to identify threats. Regular updates are necessary to ensure protection against new and evolving malware.

---

## 3. Virtual Private Network (VPN)

### Description  
A VPN encrypts internet traffic and routes it through a secure server to protect user privacy.

### Security Concept  
- Encryption  
- Privacy protection  

### Analysis  
VPNs prevent attackers from intercepting data on public networks and hide the user’s IP address. However, users must trust the VPN provider to handle their data securely.

---

## 4. Password Manager

### Description  
A password manager securely stores and generates strong passwords for multiple accounts.

### Security Concept  
- Credential management  
- Authentication security  

### Analysis  
Password managers reduce the risk of password reuse and help users maintain strong, unique passwords. They enhance security but require strong master password protection.

---

## 5. Multi-Factor Authentication (MFA)

### Description  
MFA requires users to provide two or more verification factors to access an account.

### Security Concept  
- Strong authentication  
- Defense in depth  

### Analysis  
MFA significantly reduces the risk of unauthorized access, even if passwords are compromised. It combines multiple authentication factors such as something you know (password) and something you have (OTP or device).

---

## Reflection

Online security tools play a critical role in protecting users and systems from cyber threats. While each tool addresses different aspects of security, their combined use creates a layered defense strategy that significantly improves overall cybersecurity.
