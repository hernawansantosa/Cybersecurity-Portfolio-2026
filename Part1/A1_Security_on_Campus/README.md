A1 — SafeZone Emergency App (UWA)

Objective
To analyse how SafeZone enhances campus security at UWA.

Overview
SafeZone is UWA’s emergency app that connects students directly to campus security using real-time location and alert features.

Key Features
1. One-tap emergency call
2. GPS location sharing
3. Wellbeing assistance
4. Incident reporting

Security Concepts
1. Authentication (university login required)
2. Availability (immediate access to help)
3. Confidentiality (secure transmission of location data)
4. Incident Response (faster, coordinated response)
5. Defense in Depth (digital layer supporting physical security)

Reflection
SafeZone improves response time by combining mobile technology with campus security operations.
However, its effectiveness depends on smartphone access and network availability. 
It strengthens campus safety as part of a layered security approach.
